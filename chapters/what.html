What?

A query language for your API

  • Ask & get exactly what you need
  • Fetch many resources in a single resquest
  • Describe "schema" with type system (types & fields)
  • Powerful toolkit (GraphiQL), GraphCool, Apollo, & Relay
  • Evolve your API without versions

Use Case

Compelling use case for companies with legacy infrastructures and many different APIs

GraphQL can be used to unify existing systems and hide complexity of data fetching logic
